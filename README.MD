## Install

```bash
npm i -D eslint-config-reasonable-react
```

## Usage

You can just extend it on your `.eslintrc` file:

```javascript
 {
   'extends': 'eslint-config-reasonable-react'
 }
```

## Extends
```javascript
[
  'standard',
  'standard-jsx',
  'eslint:recommended',
  'plugin:react/recommended'
]
```

## Plugins
```javascript
[
  'promise',
  'react',
  'standard'
]
```
